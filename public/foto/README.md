# Istruzioni per le immagini

Carica in questa cartella (`public/foto/`) i 192 fotogrammi per l'animazione canvas.

Assicurati che i file siano nominati esattamente in questo modo:
- `frame_000_delay-0.041s.jpg`
- `frame_001_delay-0.041s.jpg`
- ...
- `frame_191_delay-0.041s.jpg`

Una volta caricati, il sistema li rileverà automaticamente e l'animazione prenderà vita!
